## useActiveAuction

Fetches data and provides functionality for users to bid on an NFT in an active auction.

```ts copy
import { useActiveAuction } from "@public-assembly/dao-utils";
```

  <Tabs items={["Usage", "Return Values", "Configuration"]} storageKey="selected-pkg-manager" defaultIndex="0">
    <Tab>
      ```ts copy
      import { useActiveAuction } from "@public-assembly/dao-utils";
      
      function App() {
        const tokenAddress = '0x123456...';
        const {
          auctionData,
          totalSupply,
          createBid,
          createBidSuccess,
          createBidLoading,
          createBidError,
          createBidTx,
          bidAmount,
          updateBidAmount,
          isValidBid
        } = useActiveAuction(tokenAddress);
      ```
    </Tab>
    <Tab>
      ```
      | Return | Type | Description
      | -- | -- | -- |
      | auctionData | Object | Containing data of the active auction such as tokenId, address, metadata, duration, endTime, highestBidder, highestBidPrice, minBidIncrement, minBidAmount, and reservePrice.
      | totalSupply | Number | The total supply of the NFT Contract.
      | isValidBid | Boolean | Indicating if the bid amount is valid.
      | createBidLoading| Boolean | Indicating if creating a bid is in
      ```
    </Tab>
